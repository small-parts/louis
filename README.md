# Louis (hex viewer)

*为什么叫这个名字，历史有比较有名的人：路易十六，嗯嗯很对，十六*

最近我在学习汇编语言，以及准备想要重新做个os出来。

所以，我需要一个hex viewer。

当然你可能会问不是有`hexdump`，或者rust写的`hexyl` 嘛。

但是，我要告诉你的是，自己打造的工具才是最趁手的。


安装:

```bash
git clone --depth 1 -b master https://github.com/small-parts/louis.git
cd louis
cargo install --path .
```
